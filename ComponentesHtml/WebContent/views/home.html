Descripcion

angular-breadcrumb es un módulo para AngularJS, que genera un breadcrumb para cualquier pagina de tu aplicaci&ocaute;n. Esta basado en UI-router framework que maneja una jerarquía de pasos.

Observar el ejemplo para ver como trabaja.

Características principales

  • Construye un breadcrumb con un step por cada paso en la jerarquía,
  • Muestra una etiqueta por cada paso y permite que angular haga binding a este,
  • Trabaja con minima configuración,
  • Permite template personalizado (El template por defecto es Bootstrap 3 breadcrumb template).
